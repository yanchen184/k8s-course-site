## 課程簡介（Udemy / 網站用）

### 開頭 Hook（前 3 行決定學員要不要往下看）

你會 Docker 了，一台機器上跑得很好。但有一天你的容器變成幾十個、機器變成好幾台、凌晨三點機器掛了你被電話叫醒手動重建容器——這時候你就知道，Docker 不夠了。

這堂課不是從「Kubernetes 是什麼」開始的。而是從「你會遇到什麼問題」開始的。

### 教學方法（差異化賣點）

**因果鏈教學法** — 市面上的 K8s 課程大多是「Pod 是什麼、Service 是什麼、Deployment 是什麼」的概念介紹式。這堂課不一樣。

這堂課的每一個概念，都是因為**上一步出了問題**才引入的：

- Pod 跑起來了，但 IP 會變 → 所以需要 **Service**
- Service 可以連了，但地址 192.168.1.100:30080 太醜 → 所以需要 **Ingress**
- 設定寫死在 Image 裡，換環境要重 build → 所以需要 **ConfigMap**
- 密碼放 ConfigMap 誰都看得到 → 所以需要 **Secret**
- 容器掛了資料消失 → 所以需要 **Volume**
- 只有一個 Pod，掛了就停服務 → 所以需要 **Deployment**

一條因果鏈，從第一個問題串到最後一個概念。你不只知道「是什麼」，你知道「為什麼」。

### 課程結構

**四堂課，每堂 7 小時，28 小時完整學習路徑：**

| 堂次 | 主題 | 你學完之後能做什麼 |
|------|------|------------------|
| 第四堂 | K8s 全貌 + Pod | 會在 K8s 上跑容器，會讓 K8s 幫你管一群 |
| 第五堂 | Deployment + Service | 會讓瀏覽器連到 K8s 裡的服務 |
| 第六堂 | Ingress + 配置 + 持久化 | 會部署帶資料庫的完整網站 |
| 第七堂 | 生產就緒 + 總複習 | 會從零獨立建一整套生產系統 |

**每堂課的節奏：**
- 概念影片（因果鏈，不是念課本）
- 實作示範（跟著做就會）
- 學員練習時間（動手做，不只是看）
- 回頭操作（沒跟上的人在這裡補回來）

### 適合誰

- 已經會 Docker（docker run、Docker Compose），想進入 K8s 的工程師
- 看過 K8s 教學但「看完還是不知道為什麼要學這些東西」的人
- 想在履歷上加上 Kubernetes 技能的後端/DevOps 工程師

### 不適合誰

- 完全沒用過 Linux 和 Docker 的人（建議先上前三堂）
- 已經在生產環境操作 K8s 的資深 SRE（這堂課是入門到中階）

### 學完你會得到

- 能在 K8s 上部署、擴縮、更新、回滾一個完整的服務
- 能排除 Pod 的常見錯誤（ImagePullBackOff、CrashLoopBackOff）
- 能用 Service + Ingress 讓外部使用者用域名連到你的服務
- 能用 ConfigMap / Secret 管理設定和密碼
- 能用 PV/PVC 讓資料不會因為容器重啟而消失
- 能用 Probe + Resource limits + HPA 讓服務在生產環境穩定運行
- 能用 RBAC + NetworkPolicy 做基本的安全管控
- 能從零獨立建一整套 K8s 生產系統

## 封面設計建議

### 風格方向

**不要用**：
- K8s logo 放大 + 「Kubernetes 入門」文字（太無聊，跟其他課一樣）
- 滿滿的技術名詞堆疊（學員看不懂就滑走了）
- 深藍色背景 + 白色文字（太制式）

**要用**：
- 因果鏈的視覺元素（箭頭串連的概念方塊）
- 對比畫面：左邊混亂（Docker 單機、手動管理、凌晨三點被叫醒）vs 右邊整齊（K8s 自動管理）
- 問題 → 解法的視覺流

### 封面方案

**方案 A：因果鏈視覺**
- 深色背景（#0f172a slate-900）
- 中間一條因果鏈從左到右：
  `Docker 扛不住 → Pod → Service → Ingress → ... → 生產就緒`
- 每個節點用不同顏色的方塊
- 上方大標題：「Docker 扛不住了，然後呢？」
- 下方副標題：「因果鏈教學法 · 102 支影片 · 繁體中文」

**方案 B：問題對比**
- 左半邊（紅色調）：一個工程師凌晨三點被電話叫醒的插畫/icon
- 右半邊（綠色調）：K8s 自動修復、自動擴縮的示意圖
- 中間箭頭：「→ 這堂課教你怎麼從左邊走到右邊」

**方案 C：簡潔文字型**
- 深色背景
- 大字：「為什麼需要 K8s？」
- 小字：「不教你是什麼，教你為什麼」
- K8s 船舵 icon 小小放在角落

### 配色建議

| 用途 | 顏色 | Hex |
|------|------|-----|
| 背景 | Slate 900 | #0f172a |
| 主標題 | White | #ffffff |
